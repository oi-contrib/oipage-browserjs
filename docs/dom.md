# 元素操作

## appendTo

把当前维护的结点加到目标结点内部的结尾：

```js
bjs.appendTo(target: any, context?: any): this
```

## prependTo

把当前维护的结点加到目标结点内部的开头：

```js
bjs.prependTo(target: any, context?: any): this
```

## afterTo

把当前维护的结点加到目标结点之后：

```js
bjs.afterTo(target: any, context?: any): this
```

## beforeTo

把当前维护的结点加到目标结点之前：

```js
bjs.beforeTo(target: any, context?: any): this
```

## remove

从页面中删除当前维护的结点：

```js
bjs.remove(): this
```
